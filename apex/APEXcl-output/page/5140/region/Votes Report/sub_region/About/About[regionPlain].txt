Aditional Information:<br>
<ul style="list-style-type:circle">
    <li>ESID: corresponds to the Session ID.</li>
    <li>SESSION_NUM: Public number for the session.</li>
    <li>SESSION: name of the session.</li>
    <li>VOTER: name of the user who voted.</li>
    <li>UID: ID of the user who voted.</li>
    <li>VOTE_TYPE: type of the vote.</li>
    <li>VOTEVAL: value of the vote.</li>
    <li>COMMENT: comment added by the user who voted.</li>
</ul>
